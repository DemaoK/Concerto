The Symphony Zoom-in Simulation Suite
=====================================

This is the documentation page for the Symphony zoom-in simulation suite. It
contains information on the simulations, documentation for a Python library for
working with simulation data, and documentation for the pipeline that creates
that data.

.. note::
   This project is under active development.
