``symlib`` Info
==================================

``symlib`` is a data analysis library initially created for the :doc:`Symphony <simulations>` suite of simulations. The library uses custom file formats and functions to reduce the size of files, to speed up common common tasks, and to make some common tasks easier. ``symlib`` also allows you to download Symphony and similar data sets.
